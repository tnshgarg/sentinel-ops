# 8. Deterministic Grader Design

DO NOT use LLM-as-judge.

Use rubric scoring.

```python
class IncidentGrader:
    def evaluate(self, gt, action_history):
        score = 0.0

        if gt.correct_camera in action_history:
            score += 0.2

        if gt.correct_frame_detected:
            score += 0.2

        if gt.correct_risk_level:
            score += 0.3

        if gt.correct_escalation:
            score += 0.3

        return min(score, 1.0)
```

This maximizes reproducibility.
